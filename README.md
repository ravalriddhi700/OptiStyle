#  OptiStyle

OptiStyle is an online eyewear shopping website inspired by platforms like Lenskart. This project was developed using PHP and MySQL as part of my MCA academic work. It allows users to browse eyewear products, view product details, place orders, and manage their accounts through a simple and user-friendly interface.

---

## Project Overview

The main goal of this project is to provide an online platform where customers can explore different types of eyewear, including spectacles, sunglasses, and lenses. The project also includes an admin panel for managing products, users, and orders.

---

## Features

### User Module
- User Registration and Login
- Browse Eyewear Products
- Search Products
- View Product Details
- Add Products to Cart
- Checkout
- View Orders
- Manage Profile

### Admin Module
- Admin Login
- Add, Update and Delete Products
- Manage Categories
- Manage Customers
- View Orders
- Manage Website Content

## How to Run the Project

1. Copy the project folder into the `wamp64/www` directory.
2. Start Apache and MySQL using WAMP Server.
3. Import the SQL database from the `DATABASE` folder into phpMyAdmin.
4. Open your browser and visit:

```
http://localhost/optistyle
```
